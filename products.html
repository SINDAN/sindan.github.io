SINDAN Project

Simple Integrated Network Diagnosis And Notification

Overview

Sindan システムはネットワーク状態の診断をする診断ノード、各診断ノードからの診断結果を収集し、結果を表示する収集・可視化サーバから構成されています。

診断ノード

診断ノードは、下記のプラットフォームで動作します。
  • Raspberry PI(ver.1 - ver.3)
  • Linux(Debian / Ubuntu)
  • Windows7/8/10(PowerShell 実行環境が必要です)
  • MacOS X
  • Android(開発中)

Rasbperry PI版のソフトウェアは bootable イメージで、その他のプラットフォームでは実行型ファイルとして配布しています。

収集・可視化サーバ

本システムはノードからの診断結果の受け渡しに fluentd を利用しています。収集した結果はサーバ上で蓄積され、結果を Web で確認することができます。

収集・可視化サーバは Linux 上で動作します。また、Docker コンテナも用意していますので、インストールすることなく利用することも可能です。
