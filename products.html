Overview

Sindan システムはネットワーク状態の診断をする診断ノード、各診断ノードからの診断結果を収集し、結果を表示する収集・可視化サーバから構成されています。

診断ノード

診断ノードは、下記のプラットフォームで動作します。

  • Raspberry PI(ver.1 - ver.3)
  • Linux(Debian / Ubuntu)
  • Windows7/8/10(PowerShell 実行環境が必要です)
  • MacOS X
  • Android(開発中)

Rasbperry PI版のソフトウェアは bootable イメージで、その他のプラットフォームでは実行型ファイルとして配布しています。

収集・可視化サーバ

本システムはノードからの診断結果の受け渡しに fluentd を利用しています。収集した結果はサーバ上で蓄積され、結果を Web で確認することができます。

収集・可視化サーバは Linux 上で動作します。また、Docker コンテナも用意していますので、インストールすることなく利用することも可能です。

イメージ取得・書き込み

下記ダウンロードリンクより Raspberry PI のイメージをダウンロードし、zip ファイルを解凍してください。書き込み用の SD カードを用意してください。最低でも 4G の容量が必要です。ダウンロードしたファイルを、bootable イメージを書き込めるツール等を利用して SD カードに書き込みをしてください。Windows であればWin32 Disk Imagerなどが利用できます。

設定

イメージを書き込んだ SD カードを確認すると、いくつかのフォルダが見えます。その中の /etc フォルダに設定ファイルがあります。

  • sindan.conf
    • sindan.sh用の設定ファイル
  • wpa_supplicant.conf
    • 測定する ssid の設定ファイル
  • crontab
